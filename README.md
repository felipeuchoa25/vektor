# 🎯 Vektor

Sistema de análise de campanhas, garimpagem especial, criador de campanhas e presells para Google Ads.

## Versões

| Arquivo | Descrição |
|---------|-----------|
| vektor_v8.html | Versão atual — inclui Criador de Campanhas com scoring e integração Google Ads API v18 |
| vektor_v7.html | Copy Engine, Deep Scan, Finance, YouTube tracking |

## Como usar

Abra qualquer arquivo `.html` diretamente no navegador. Nenhuma instalação necessária.
